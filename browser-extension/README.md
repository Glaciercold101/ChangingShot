# Browser Extension - ChangingShot

## Installation

### For Edge Browser:
1. Open Edge and go to `edge://extensions/`
2. Enable "Developer mode" (top-right toggle)
3. Click "Load unpacked"
4. Select this `browser-extension` folder
5. Extension will appear in your toolbar

### For Chrome:
1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (top-right toggle)
3. Click "Load unpacked"
4. Select this `browser-extension` folder
5. Extension will appear in your toolbar

## Features

- 📍 **Click to Select**: Hover over elements and click to select monitoring area
- 🖱️ **Right-Click Menu**: Quick access via context menu
- 💾 **Auto-Save**: Selected areas are saved to browser storage
- 📡 **Sync**: Areas sync to your monitoring dashboard

## Files

- `manifest.json` - Extension configuration
- `popup.html/css/js` - Popup UI
- `content.js` - Webpage interaction
- `background.js` - Event handling

## Usage

1. Click the ChangingShot icon in your toolbar
2. Click "Select Area on Website"
3. Hover over the element you want to monitor
4. Click to confirm selection
5. Area is now added to your monitor list!
